# 校园招聘信息管理系统 python1023

## 项目概述

校园招聘信息管理系统是基于 Spring Boot 和 Vue 技术栈构建的一套高效的信息管理平台。该系统旨在为校园招聘提供便捷的管理与交流环境，服务对象包括管理员、学生、企业和导师。

## 技术栈

- 后端：Java（Spring Boot）
- 前端：Vue
- 数据库：MySQL

## 运行环境

- Java版本：JDK 1.8
- Node版本：13.x（如不修改项目可无需安装）
- Python版本：2.7（如不修改项目可无需安装）
- IDE类型：IntelliJ IDEA 或 Eclipse
- 数据库：MySQL（5.x 或 8.x）
